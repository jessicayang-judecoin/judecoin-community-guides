## Notice

The CLI version mentioned in some examples is v3.1.2. If Judecoin releases a newer CLI version, please always follow the latest official version and update the download link, file name, folder name, and version number accordingly.

## Security Reminder

Never share wallet seed phrases, wallet private keys, Service Node private keys, AWS .pem files, or server login information.
